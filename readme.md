# Labs Portfolio

Detta repository innehåller laborationer från kursen, organiserade per vecka. Syftet är att samla allt kursarbete på ett ställe och fungera som en portfolio som växer under kursens gång.

## Struktur

Laborationerna är grupperade per vecka. Varje laboration kan innehålla HTML, CSS, JavaScript eller andra filer som krävs.

### Vecka 1
- [Laboration 1-a](1a/index.html) – Skapa din första webbsida med HTML och CSS.
- Laboration 1-b – Ej publicerad än.
- Laboration 1-c – Ej publicerad än.

### Vecka 2
- Laboration 2-a – Ej publicerad än.
- Laboration 2-b – Ej publicerad än.

### Vecka 3
- Laboration 3-a – Ej publicerad än.
- Laboration 3-b – Ej publicerad än.

## Startsida

En **startsida** finns i `index.html` direkt i labs-mappen. Den innehåller en lista över alla laborationer med länkar, titlar och kort beskrivning.

## Styling

All gemensam styling ligger i `global.css`, som kan återanvändas av alla laborationssidor.
